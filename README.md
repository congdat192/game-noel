# Game "Catch & Win - Bắt Quà Noel"

Đây là mã nguồn cho mini-game "Catch & Win", một ứng dụng web được thiết kế để thu thập khách hàng tiềm năng (qua SĐT) và phát voucher mua hàng thông qua một trải nghiệm chơi game vui vẻ, đơn giản.

## Các Công nghệ chính

*   **Frontend:** React (Vite)
*   **Game Engine:** Pixi.js
*   **Backend & Database:** Supabase
*   **Authentication:** SMS OTP

## Hướng dẫn Cài đặt & Triển khai

Toàn bộ hướng dẫn chi tiết về cách thiết lập môi trường, cấu hình backend trên Supabase và triển khai dự án lên Vercel đều nằm trong file `DEPLOYMENT.md`.

Vui lòng đọc kỹ file **[DEPLOYMENT.md](./DEPLOYMENT.md)** để bắt đầu.
